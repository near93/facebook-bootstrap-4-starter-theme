## About: Bootstrap Starter Theme
## Description: A starter theme based on Facebook's color scheme.

Currently using Bootstrap 4

## Tools to use
GruntJS - see more here (http://gruntjs.com/getting-started)

## Getting Started with GruntJS
- Install [node.js](https://nodejs.org)
- Using the command line, navigate to your theme directory. 
- Run npm install
- Run grunt to confirm everything is working
- Run grunt watch 

## Working with Sass files
- To override global variables copy the code you'd want to change from the '_variables' and 
copy to '_custom' removing the !default code.
